# MemoryConnect

A cognitive wellness app that generates personalized stories for people with memory loss.

## About the Project

MemoryConnect helps caregivers create personalized stories that incorporate elements from a person's life history, using cognitive wellness principles and Eastern philosophical elements to provide therapeutic benefits.

### Key Features

- **Profile Management**: Securely store basic information about clients while respecting privacy
- **Story Generation**: Create customized stories with cognitive wellness elements
- **Guided Delivery**: Interactive guidance for effective story presentation
- **Assessment Tracking**: Monitor engagement and effectiveness over time

## Current Status

This repository contains the initial concept and UI components for the MemoryConnect app. It's currently in prototype stage and not yet a fully functional application.

### Technologies

- React.js for the frontend UI
- Tailwind CSS for styling
- (Future) Integration with AI services for story generation

## Development Roadmap

- [x] Initial UI concept

## Getting Started

### Prerequisites

- Node.js and npm installed
- Basic knowledge of React

### Installation

1. Clone the repository
   ```
   git clone https://github.com/YOUR_USERNAME/memory-connect-app.git
   ```
2. Install NPM packages
   ```
   npm install
   ```
3. Start the development server
   ```
   npm start
   ```
